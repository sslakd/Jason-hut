# Bài 14: MACD

3 thành phần: MACD Line (EMA12 - EMA26), Signal Line (EMA9 của MACD), Histogram.

---

## Cách dùng

**MACD Cross:**
- MACD cắt lên Signal → mua
- MACD cắt xuống Signal → bán
- Cross gần zero line càng mạnh

**Histogram:**
- Cột xanh to dần → momentum tăng
- Cột xanh nhỏ dần → momentum yếu
- Histogram đổi màu → tín hiệu sớm

**Zero line:**
- MACD > 0 → bullish
- MACD < 0 → bearish

## Kết hợp MACD + MA

1. Giá trên EMA20 → chỉ nhìn tín hiệu mua
2. MACD cắt lên Signal → trade
3. Histogram tăng → giữ
4. MACD cắt xuống Signal → chốt lời

Ngược lại cho downtrend.
